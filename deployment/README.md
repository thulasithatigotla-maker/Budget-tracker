# Finance Manager App 
 
"This package contains the Finance Manager app with facial recognition features." 
 
"## Files included:" 
"- index.html: Main application" 
"- assets/: Application assets (CSS, JS, images)" 
"- face_id_demo.html: Standalone facial recognition demo" 
"- mobile_access_qr.html: QR codes for mobile access" 
"- start_server.bat: Simple HTTP server to run the app locally" 
 
"## How to run:" 
"1. Double-click start_server.bat" 
"2. Open your browser and go to http://localhost:8080" 
"3. For facial recognition demo, open face_id_demo.html directly in your browser" 
 
"## Features:" 
"- Full financial management application" 
"- Facial recognition authentication (Face ID)" 
"- Biometric authentication using Web Authentication API" 
"- PIN-based authentication as fallback" 
"- Offline mode support"
